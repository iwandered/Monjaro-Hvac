# 星越L 2021-2023款 环境面板

针对吉利星越L车型（2021-2023款）开发的银河OS增强环境面板控制应用，完全使用DeepSeek、通义千问、Antigravity （Gemini 3.0）生成代码。

![Monjaro-DataDim](https://github.com/iwandered/Monjaro-Hvac/blob/main/hvac.png)

---

## 主要功能


---目前无法实现风量控制、设定温度显示和控制以及与风量控制和温度控制相关的功能。
目前实现主副驾驶座椅加热，风向一键调节和组合调节，AC开关，AUTO开关，内外循环切换，空气净化g-clean一键开关，车内外温度传感器显示，手动、自动风量显示，副驾驶有人时显示欢迎文字，点击左下角Home键返回默认桌面，点击hvac打开原车空调，点击seat打开原车座椅设置界面

## 使用说明

- 将ecarx.jar和ecarx.adaptapi.jar，经dex2jar处理后放入/app/libs/中。
- Android Studio编译调试.
- 工具自启动需要授予无障碍权限（使用应用管家授予权限）  
   *注意：出于安全考虑ecarx.jar和ecarx.adaptapi.jar不随源码提供请自行从车机提取*
